## About The Project

This program allows a user to sell tickets various types of tickets such as: general, advance and student, search for purchased tickets,
manage clients performing shows, verify credit cards and implement user-interface

Here's how:
* The program uses a super/sub class stucture for the tickets 
* Tickets sold are saved to the disk



### Built With

The program was built using Java
